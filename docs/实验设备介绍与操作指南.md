# 实验设备介绍与操作指南

## 一、设备总览

本实验采用力板系统和可穿戴惯性传感器系统两套设备，通过同步电缆实现时间对齐，以100Hz采样率同步采集受试者坐站转换（STS）过程中的地面反力和运动学数据。

| 设备 | 型号 | 厂商 | 数量 | 用途 |
|------|------|------|------|------|
| 力板 | BP400600-OP-1000 | AMTI（美国） | 2块 | 采集垂直地面反力（GRF） |
| 信号放大器 | Optima | AMTI（美国） | 1套 | 放大力板模拟信号 |
| 无线惯性传感器 | MTw | Xsens Technologies BV（荷兰） | 2个 | 采集3轴加速度+3轴角速度 |
| 无线接收器 | Awinda Station | Xsens Technologies BV（荷兰） | 1个 | 接收IMU无线数据 |

## 二、设备详细介绍

### 2.1 AMTI 力板（BP400600-OP-1000）

**功能**：测量人体与地面/座椅之间的三维交互力（垂直力、前后力、左右力）。

**在本实验中的作用**：
- **力板1**（放置在脚下）：测量脚-地面垂直反力，用于定义STS各相位边界
- **力板2**（放置在座椅下）：测量臀部-座椅垂直反力，用于判断臀部离座时刻

**关键参数**：
- 测量范围：垂直方向±4448N，水平方向±2224N
- 尺寸：400mm × 600mm
- 精度：±0.1% 满量程
- 采样率：可达1000Hz（本实验设为100Hz）

**数据作为标注的金标准**：力板数据用于精确划分5个STS相位的边界，具体阈值为：
- 初始坐姿 → 屈曲动量：GRF开始变化
- 屈曲动量 → 动量转移：座椅力降至0N（臀部离座）
- 动量转移 → 伸展：脚下GRF达到峰值
- 伸展 → 稳定站立：GRF等于体重并保持稳定

### 2.2 Optima 信号放大器

**功能**：将力板输出的微弱模拟电信号放大至可被采集系统识别的电压范围。

**在本实验中的作用**：
- 连接力板与数据采集系统
- 提供信号调理（滤波、放大）
- 通过同步电缆与Xsens系统实现时间同步

### 2.3 Xsens MTw 无线惯性传感器

**功能**：集成三轴加速度计、三轴陀螺仪和三轴磁力计的微型可穿戴惯性测量单元（IMU）。

**在本实验中的佩戴位置**：
- **传感器1**：佩戴于 **L5 腰椎位置**（背部下方）
  - 用途：采集坐站过渡期间的躯干运动数据，作为CNN-BiLSTM-Attention模型的主要输入
- **传感器2**：佩戴于 **大腿外侧**
  - 用途：通过欧拉角阈值区分初始坐姿和稳定站立两个静态相位

**关键参数**：
- 加速度计范围：±160 m/s²（±16g）
- 陀螺仪范围：±2000 °/s
- 磁力计范围：±1.9 Gauss
- 采样率：最高120Hz（本实验100Hz）
- 输出：原始数据 + 内置融合算法计算的四元数
- 通信方式：2.4GHz无线（通过Awinda Station接收）
- 重量：约27g
- 尺寸：47mm × 30mm × 13mm

**输出数据格式**（每帧10列）：
| 列号 | 内容 | 单位 |
|------|------|------|
| 1 | 时间戳 | - |
| 2-4 | 3轴加速度（X/Y/Z） | m/s² |
| 5-7 | 3轴角速度（X/Y/Z） | rad/s |
| 8-11 | 四元数（q0/q1/q2/q3） | - |

### 2.4 Awinda Station 接收器

**功能**：Xsens无线传感器系统的基站，负责接收和转发IMU数据。

**在本实验中的作用**：
- 接收2个MTw传感器的无线数据
- 通过USB连接至PC端的MT Manager软件
- 提供与力板系统的硬件时间同步接口

## 三、实验操作指南（How-to）

### 3.1 实验前准备

#### 步骤1：设备布置
```
1. 在实验区域放置两块力板：
   - 力板1：放在地面上，受试者站立时双脚踩在上面
   - 力板2：放在座椅下方或嵌入座椅，用于测量臀部压力
   
2. 将力板通过数据线连接至 Optima 信号放大器

3. 将 Awinda Station 放置在受试者2-3米范围内
   通过 USB 连接至实验电脑

4. 准备一把无扶手、无靠背的座椅
   椅高设为受试者膝高的85%
```

#### 步骤2：软件配置
```
1. 打开 Xsens MT Manager 软件
   - 确认 Awinda Station 已连接
   - 设置采样率为 100Hz
   - 配置同步输出信号

2. 打开 AMTI 力板数据采集软件
   - 设置采样率为 100Hz
   - 配置触发/同步模式（通过同步电缆与Xsens同步）
   
3. 连接同步电缆
   - 将力板系统的同步输出连接至 Awinda Station 的同步输入
   - 确保两套系统在同一时钟下采集
```

#### 步骤3：传感器佩戴
```
1. 使用约束带将 IMU传感器1 固定于受试者 L5 腰椎位置
   - 传感器X轴朝上，Y轴朝前
   - 确保紧贴皮肤，无松动

2. 使用约束带将 IMU传感器2 固定于受试者大腿外侧
   - 传感器方向与大腿骨长轴对齐
   - 确保不影响受试者正常运动

3. 在 MT Manager 中校准传感器
   - 执行磁力计校准（8字形转动）
   - 确认四元数输出正常
```

### 3.2 实验执行

#### 步骤4：受试者准备
```
1. 受试者签署知情同意书

2. 记录受试者基本信息：
   - 身高、体重、年龄、性别
   - 确认无肌肉骨骼疾病或严重神经损伤

3. 指导受试者就坐初始姿势：
   - 臀部接触座椅
   - 眼睛平视前方
   - 双臂放松交叉于胸前
   - 躯干竖直
   - 大腿与地面水平
   - 小腿与地面垂直
   - 双脚平行，间距与肩同宽
```

#### 步骤5：数据采集
```
1. 同时启动力板和IMU数据记录

2. 实验员发出"站起来"口令

3. 受试者以自然速度站起
   - 从初始坐姿开始
   - 自然站立直至身体稳定
   - 受试者自觉稳定后发出"停止"口令

4. 间隔2秒后，受试者恢复初始坐姿

5. 重复上述过程 5次

6. 停止数据记录

7. 导出数据：
   - Xsens数据导出为 .txt 文件（tab分隔，前6行为头信息）
   - 力板数据导出为对应格式文件
```

### 3.3 数据处理

#### 步骤6：IMU数据预处理
```
对应代码：IMU dataprocessing/data_process.m

1. 读取 Xsens .txt 原始数据文件
2. 提取加速度（列2-4）、角速度（列5-7）、四元数（列8-11）
3. 利用四元数将加速度从传感器坐标系旋转到世界坐标系
4. 减去重力分量 [0, 0, 9.8]，得到线性加速度
5. 应用12阶低通巴特沃斯滤波器（截止频率1.7Hz）
6. 输出滤波后的垂直线性加速度和角速度
```

#### 步骤7：相位标注
```
对应代码：phase segmentaion and build dataset/lable.m

1. 加载力板垂直地面反力数据
2. 基于阈值条件自动标注5个STS相位的边界
3. 人工校验自动标注结果
4. 生成带标签的数据样本
```

#### 步骤8：构建数据集
```
对应代码：phase segmentaion and build dataset/ 目录

1. 批量重命名和格式转换（1_rename.m, 2_TOmat.m）
2. 多受试者数据拼接（3_pijie_.m）
3. 数据增强：添加高斯噪声 + 时间伸缩（DataEnhancement.m）
4. 提取6维特征子集（tiqu_6.m）
5. 按7:3比例划分训练/测试集（4_final.m）
6. 保存为 STS-PD dataset.mat
```

#### 步骤9：模型训练与评估
```
对应代码：enhance/train.py（加强版，支持MPS GPU）

cd enhance
.venv/bin/python train.py --model cnn_bilstm_attention --epochs 100

# 或运行所有模型对比
.venv/bin/python train.py --all --ml
```

## 四、注意事项

### 安全与伦理
- 实验需获得伦理委员会批准
- 受试者需签署知情同意书
- 佩戴传感器时注意受试者舒适度
- 确保实验区域安全，防止受试者跌倒

### 数据质量
- 每次实验前校准力板零点
- 检查IMU传感器电量充足（>50%）
- 确认同步信号正常（力板与IMU时间戳对齐）
- 排除受试者非典型动作数据（如中途停顿、手扶椅子等）

### 设备维护
- 力板定期校准（每6个月或频繁使用后）
- IMU传感器使用后充电，存放于干燥环境
- 约束带使用后清洁消毒
- 同步电缆避免弯折，检查接口无松动
